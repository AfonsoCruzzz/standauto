# 🚗 Stand Automóvel - MVP

Site para gestão e apresentação de viaturas de um stand automóvel.

## 🛠 Stack Tecnológica

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL + Prisma ORM
- **Deploy**: Vercel

## 🚀 Começar a Desenvolver

### Pré-requisitos
- Node.js 18+
- PostgreSQL
